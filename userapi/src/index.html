Welcome to Tristan's and Apolline's userapi

Complete API documentation has been created using Swagger

Here's a table reviewing all available API methods and the path assigned to these methods:

Method Path Usage
POST /user/ use this method on given path to add a new user to the REDIS DB. In the post request send a JSON body containing following information: {"username" = "", "firstname" = "", "lastname"=""}
GET /user/:username use this method on given path to retrieve a specific user (giving his username in URL) from the REDIS DB.
DELETE /user/:username use this method on given path to delete a specific user (giving his username in URL ) from the REDIS DB.
GET /user/keys/ use this method on given path to retrieve all keys present in the REDIS DB.
GET /api-docs/ obtain complete documentation of the API

IMPORTANT: This application requires a REDIS DB to work perfectly. If you are using this application on AZURE or in a Docker container not linked with a REDIS container, API methods won't work for you.
